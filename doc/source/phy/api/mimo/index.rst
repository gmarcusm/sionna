Multiple-Input Multiple-Output (MIMO)
=====================================

.. currentmodule:: sionna.phy.mimo

This module provides components for MIMO signal processing in communication systems.
It includes :doc:`stream_management` for configuring multi-user MIMO setups,
:doc:`precoding` functions for transmit beamforming,
:doc:`equalization` for linear receive processing,
:doc:`detection` algorithms for symbol recovery,
and :doc:`utility` functions for common MIMO operations.
