Utility Functions
=================

.. currentmodule:: sionna.phy.utils

This module provides general-purpose utility functions used throughout Sionna.
It includes :doc:`linear_algebra` operations for matrix computations,
:doc:`metrics` for evaluating communication system performance,
:doc:`numerics` for numerical operations and special functions,
:doc:`tensors` for tensor manipulation and reshaping,
:doc:`random` for random number generation,
:doc:`plotting` for visualization helpers,
and :doc:`misc` for other commonly used functions.
